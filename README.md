=== BASH PROJECTS ===

#1. Infrastructure Health Monitoring Script

A Bash script that performs basic infrastructure health checks such as CPU usage, memory usage, disk usage, and network connectivity.  
The script displays results in real time and saves timestamped logs for auditing and troubleshooting.

===============

#2. Log Cleanup Script

A Bash script that performs log cleanup from '/var/log' which deletes logs older than 7 days.

================

#3. Log Rotation Script

This Bash script performs automated log rotation for log files. Itarchives .log files, compresses them to save disk space, and removes old archives based on a retention policy.

================

#4. Service Health Check Script

This Bash script monitors a Linux systemd service and automatically attempts to restart it if it goes down. All actions are logged with timestamps for auditing and troubleshooting.
